`GoImportsBear <https://github.com/coala/coala-bears/tree/master/bears/go/GoImportsBear.py>`_

Adds/Removes imports to Go code for missing imports.

* Go



Dependencies

Can Detect

* Missing import

Can Fix
----------

* Missing import
